コードのリンクを貼らせていただきましたので可能な範囲でご覧ください

外出状況

事務所が2階建てだったので、誰が在席中で誰が外出中かPC上でわかるようにするために作成しました。
登録時にクリック数が少ないほうがいいという要望があったので、ダイアログで登録ができるようにしました。IDを共有して使用していたのでレコードデータごとにボタンを設置してワンクリックで帰所登録できるようにしました。

相続チェックリスト

相続の初回面談時に収集してほしい書類の一覧を渡さずに後日作成してご案内していましたが、その場でチェックリスト通りに質問して 一覧表が出てくるようにしたら印刷するだけで済むので時間短縮になると思い作成しました。 ヒアリングシートにもなりますし、渡したデータもkintoneに自動的に保存が出来て喜ばれました。

GitHub

備品入出庫×備品在庫管理

自家消費や関与先への販売等、全てエクセルで管理していたのでアプリ化しました。トップ画面を開いた時に在庫が一定数を下回っていないか確認し、該当があればアラートで知らせるような機能もつけたことで 総務の担当者が在庫を見に行かなくても注文するタイミングがわかるようにしました。

GitHub

経理代行管理

経理代行をしている関与先の日々の業務の管理のため、毎月の予定を一括で登録出来るようにして、期限当日の朝にはslackに通知が届くようにしました。 業務を失念してしまったことがあり、卓上カレンダーにタスクを手書きしていると相談されたので、失念しないように期日の朝にはslackに通知がくるので担当者に大変喜ばれました。
